# **App Name**: Chhattisgarh Bandhan

## Core Features:

- User Authentication: Secure authentication using Google OAuth for easy and safe login.
- Profile Creation: Multi-step form for creating detailed user profiles, capturing personal, religious, and family background information.
- Match Suggestions: AI-powered recommendations based on mother tongue, native district, caste, and partner preferences.
- Profile Viewing: Comprehensive profile display with photo gallery, horoscope section, and match score.
- Chat Messaging: Realtime chat functionality for users to connect and communicate, including message display and image uploads.
- Admin Dashboard: Admin interface with modules for user management, profile review, agent management, and subscription tracking.
- Subscription UI: Subscription pages, benefits comparison, and receipt UI components

## Style Guidelines:

- Primary color: Warm orange (#FF9800), inspired by the vibrant culture and warmth of Chhattisgarh.
- Background color: Light tan (#F5F5DC), providing a neutral and calming backdrop to accentuate the content.
- Accent color: Earthy green (#4CAF50), complementing the orange and grounding the palette with a natural touch.
- Body and headline font: 'PT Sans' for clear readability and a modern feel.
- Use clean, modern icons from a set like 'Font Awesome' or 'Material Icons'.
- Mobile-first, responsive design with clear sections and easy navigation.
- Subtle transitions and animations for a smooth user experience.
